NLP · Sentiment Analysis
Airline Tweets Sentiment Analysis

Built a complete NLP pipeline for 14,100 airline tweets (3-class: positive / neutral / negative). Designed a custom preprocessing pipeline with TweetTokenizer — preserving negation words and emoji semantics while removing noise. Compared 7 models across traditional ML and deep learning: Logistic Regression outperformed LSTM (80.4% vs 71.4%), revealing that TF-IDF features are more effective than sequence modeling on this small, short-text dataset. Fine-tuned RoBERTa achieved the best result at 83.03%, with the strongest handling of the hardest neutral class (precision 73%). Class imbalance (63% negative) was addressed via weighted CrossEntropyLoss in RNN training.

83.03% — RoBERTa (best) 80.38% — Logistic Regression 71.39% — LSTM baseline
